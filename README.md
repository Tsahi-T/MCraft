# MCraft ⛏️

קלון MCraft שרץ **כולו בדפדפן** — Frontend בלבד, בלי שרת, בלי build, בלי קבצי נכסים (כל הטקסטורות מצוירות פרוצדורלית על Canvas).

**איך משחקים:** פותחים את `index.html` בדפדפן (נדרש אינטרנט לטעינת Three.js מ-CDN בלבד). אפשר גם להפעיל דרך GitHub Pages.

## מה יש בפנים

- 🌍 **עולם "אינסופי"** — צ'אנקים נוצרים ונטענים סביב השחקן תוך כדי הליכה. בפועל העולם הוא טורוס (מתעטף כל 1,024 בלוקים) — אשליה מושלמת של אינסוף עם זיכרון חסום.
- 🏔️ **יצירה פרוצדורלית** — ביומים (מישורים, יער, מדבר, טונדרה מושלגת, הרים, אוקיינוס, חוף), רכסי הרים, מערות ספגטי ומערות ענק, עפרות (פחם/ברזל/זהב/יהלום), עצי אלון ואשוח, קקטוסים, פרחים, קרח על אגמים קפואים.
- 🗺️ **מפה שנבנית** — מיני-מפה חיה + מפת עולם מלאה (M) שמתגלה ככל שחוקרים.
- ☀️ **מחזור יום/לילה** — שמש, ירח, כוכבים, זריחות ושקיעות כתומות, עננים נעים.
- 🧱 **בנייה והריסה** — סרגל 9 חריצים, ארון בלוקים (E), העתקת בלוק בעכבר אמצעי, חלקיקים וצלילים מסונתזים.
- 🏊 **פיזיקה** — קפיצה, ריצה, שחייה מתחת למים (עם ערפל כחול), מצב טיסה (F).
- 💾 **שמירה אוטומטית** — כל שינוי בעולם, המיקום והשעה נשמרים ב-localStorage לכל Seed.
- 💡 **תאורה מזויפת** — Ambient Occlusion לכל קודקוד + החשכת מערות לפי עומק.
- ✈️ **בסיס חיל אוויר** — מבנה קבוע אחד שמוטבע בעולם: מסלול טיסה עם פסי הפרדה ותאורת שוליים, האנגר כפול עם מטוסים בפנים, מגדל פיקוח עם קומת זכוכית ופנס אזהרה, ומטוסים חונים על המצע הבטוני.
- 📱 **תמיכה מלאה במגע** — בטלפון/טאבלט מופיעים ג'ויסטיק וירטואלי וכפתורי מגע אוטומטית.

## שליטה

| מקש | פעולה |
|---|---|
| W A S D | תנועה |
| רווח | קפיצה / שחייה / טיסה למעלה |
| Shift | ריצה |
| C | טיסה למטה |
| עכבר שמאל | שבירת בלוק |
| עכבר ימין | הנחת בלוק |
| עכבר אמצעי | העתקת הבלוק שמכוונים אליו |
| גלגלת / 1–9 | בחירת חריץ |
| E | ארון הבלוקים |
| F | הפעלת/כיבוי טיסה |
| M | מפת העולם המלאה |
| N | דילוג לבוקר |
| F3 | פאנל מידע טכני |

### מסך מגע (מזוהה אוטומטית)

| מחווה | פעולה |
|---|---|
| ג'ויסטיק שמאלי | תנועה (דחיפה עד הקצה = ריצה) |
| גרירה על המסך | סיבוב המבט |
| הקשה קצרה | הנחת בלוק בנקודת המגע |
| לחיצה ארוכה | שבירת בלוק (מחזיקים להמשך שבירה) |
| ⬆ | קפיצה / שחייה / עלייה בטיסה |
| ✈ | הפעלת/כיבוי טיסה (⬇ לירידה) |
| 🎒 / 🗺 / ⏸ | ארון בלוקים / מפה / השהיה |

אפשר לבדוק את ממשק המגע גם בדפדפן שולחני עם `?touch=1` בכתובת.

## מבנה הקוד (נועד להרחבה)

```
index.html          נקודת הכניסה + HUD
css/style.css       עיצוב
js/config.js        קבועים גלובליים (גודל עולם, מהירויות, מרחק רינדור...)
js/noise.js         רעש פרלין/ערך עם מחזוריות (מה שמאפשר טורוס חלק)
js/blocks.js        רישום סוגי הבלוקים ותכונותיהם
js/textures.js      אטלס טקסטורות מצויר על Canvas
js/worldgen.js      יצירת שטח: ביומים, גבהים, מערות, עפרות, עצים
js/airbase.js       מבנה קבוע: בסיס חיל אוויר (מסלול, האנגרים, מגדל, מטוסים)
js/world.js         אחסון צ'אנקים, גישה לבלוקים, עריכות ושמירה
js/mesher.js        המרת ווקסלים לגאומטריית Three.js (עם AO)
js/sky.js           שמיים: שמש/ירח/כוכבים/עננים/צבעים
js/effects.js       חלקיקים, סאונד מסונתז, Raycast לבלוקים
js/player.js        פיזיקת שחקן והתנגשויות
js/minimap.js       מפת החקירה
js/ui.js            סרגל, ארון בלוקים, טוסטים, דיבאג
js/touch.js         שליטה במסך מגע: ג'ויסטיק, מחוות, כפתורים
js/main.js          חיבור הכל: לולאת משחק, סטרימינג צ'אנקים, קלט
```

נקודות הרחבה טבעיות: בלוק חדש = שורה ב-`blocks.js` + טקסטורה ב-`textures.js`; ביום חדש = `worldgen.js`; מבנים (כפרים וכו') = פס נוסף ב-`genChunk`.
